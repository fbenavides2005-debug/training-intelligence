# Training Intelligence

Connected training intelligence platform for athletes and active people.

## Core Features
- Connect Apple Health / Apple Watch
- Connect WHOOP
- Sync training and recovery data automatically
- Analyze load, fatigue and readiness
- Generate next-session and weekly recommendations
- Support multiple training types through a unified data model

## Project Structure
- `mobile/` mobile application
- `backend/` API and server logic
- `ai/` prompts and recommendation logic
- `docs/` product and technical documentation

## Goal
Build an intelligent coaching system that helps users train smarter using connected performance and recovery data.
